O4HTS

  • O4HTS - Organisation for Humane Technical Standardisation
  • O4HTS - Organización para Estandarización Técnica Humana
  • O4HTS - Organização para Normatização Técnica Humana
  • O4HTS - Organisation pour la Normalisation Technique Humaine

We are a group of people who want to write open technical standards to improve the humaness of software and bureaucracy, specially in sensitive topics like personal names, gender, and race.
